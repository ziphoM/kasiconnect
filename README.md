# KasiConnect - Township Services Platform

Connecting local workers ("hustlers") with clients in townships across South Africa.

## 🚀 Features

- **For Clients**: Post jobs, browse workers, hire with packages
- **For Workers**: Apply for jobs, purchase application passes, build reputation
- **Admin Panel**: Manage users, jobs, packages, and view analytics
- **USSD Support**: Access the platform without smartphones

## 🛠️ Tech Stack

- **Frontend**: React.js, React Router, Axios
- **Backend**: Node.js, Express.js, MySQL
- **Authentication**: JWT, Bcrypt
- **File Upload**: Multer
